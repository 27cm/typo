substr()

substr(int $start, int $length) : \Wheels\Typo\type

Возвращает подстроку.

Параметры

int $start

Если start неотрицателен, возвращаемая подстрока начинается с позиции start от начала текста, считая от нуля.

                 Если start отрицательный, возвращаемая подстрока начинается с позиции, отстоящей на start символов от конца текста.
int $length

Если length положительный, возвращаемая строка будет не длиннее length символов, начиная с параметра start.

                 Если length отрицательный, то будет отброшено указанное этим аргументом число символов с конца текста.

Returns

\Wheels\Typo\type

substr_replace()

substr_replace(string $replacement, int $start, int $length)

Заменяет часть строки.

Параметры

string $replacement

Строка замены.
int $start

Если start положителен, замена начинается с символа с порядковым номером start в тексте.

                       Если start отрицателен, замена начинается с символа с порядковым номером start, считая от конца текста.
int $length

Если аргумент положителен, то он представляет собой длину заменяемой подстроки в тексте.

                       Если этот аргумент отрицательный, он определяет количество символов от конца текста, на которых заканчивается замена.

strpos()

strpos(mixed $needle, int $offset) : mixed

Возвращает позицию первого вхождения подстроки.

Параметры

mixed $needle

Если не является строкой или массивом, то приводится к целому и трактуется как код символа.

                 Если является массивом, то результатом будет ассоциативный массив с ключами:
                 'pos' - позиция первой найденной подстроки из массива needle;
                 'str' - значение первой наденной подстроки из массива needle;
int $offset

Если этот параметр указан, то поиск будет начат с указанного количества символов с начала текста.

Returns

mixed —

Возвращает позицию, в которой находится искомая строка, относительно начала текста (независимо от смещения offset).

             Возвращает FALSE, если искомая строка не найдена.

string $pattern

Параметр flags регулирует порядок вывода совпадений в возвращаемом многомерном массиве.

\Wheels\Typo\type —

Возвращает количество найденных вхождений шаблона (которое может быть и нулем) либо FALSE,
