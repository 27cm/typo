\Wheels

\Wheels Config

Класс для работы с параметрами настроек.

Объекту этого класса может быть задано описание конфигурации, включающее имена, типы, значения параметров по умолчанию и т.п.

Параметры конфигурации могут быть установлены из ассоциативного массива, конфигурационного INI файла, в том числе из отдельной секции INI файла.
